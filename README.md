# Moonshot Mikey

Stupid simple static website for a Solana meme coin.

## What to edit

- `index.html` for all text
- `styles.css` for colors and layout
- `script.js` for the small reveal and number animation

## Publish fast

1. Open `index.html` in a browser to preview it
2. Upload the folder to Netlify, Vercel, or GitHub Pages
3. Replace buttons with your real links later if you want

## Tone

This version is intentionally simple, overconfident, and not pretending to be serious crypto infrastructure.
